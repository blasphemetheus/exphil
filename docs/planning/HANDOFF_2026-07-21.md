## GOTCHA CANDIDATES (write into GOTCHAS.md when touched next)
- Netplay: mainline Slippi build needs a MAINLINE-format home
  (~/.config/slippi-dolphin/netplay-beta layout, NO `ishiiruka`
  marker). Feeding it the Ishiiruka home (~/.config/SlippiOnline*)
  crashes at online-session start with CoreTiming ScheduleEvent
  nullptr. Per-account: copy the launcher's mainline home, swap
  user.json.
- pgrep -f "<pattern>" self-matches the shell running the command when
  the pattern appears in your own argv — a `for p in $(pgrep -f ...);
  do kill` prefix in a launch command killed the launch (attempts e/g
  died instantly, no log). Verify processes with `ps -eo pid,comm`, and
  never combine a pgrep-kill with the thing you're launching.

- Microbench fusion != full-graph fusion: two block-level benches
  misattributed the r15 slowdown; only profile_train_step.exs (real
  drill graph) gave the true per-step ranking.
- Axon.build seeds param init from system_time() by default — pass
  :seed for reproducible fits (bit edifice SAETrainer).
- mix compile beside a live trainer only kills it if libexla RELINKS;
  pure-Elixir compiles squeak through — do not rely on this (got lucky
  14:1x today; the pkill that "stopped" the run had silently failed —
  VERIFY process death by PID, not by pkill exit).
